자바 폴더 밑에 존재하는 패키지

*class 문법의 용도
 - 서로 관련된 일을 하는 메소드를 관리하기 쉽게 분류할 때 사용하는 문법
   - 역할에 따라 메소드 분류
 - 데이터 타입(메모리 설계도)을 새로 정의할 때 사용
   - primitive type : 자바에서 기본으로 제공하는 최소 단위 데이터 타입
   - 사용자 정의 데이터 타입 : App 쓰임새에 맞춰 개발자가 임의로 정의한 것

*낱개 변수와 묶음 변수
 - new Member() -> name[] age[] working[]
 - class { String name, int age, boolean working }

*클래스와 인스턴스
 - 클래스 = 메모리 설계도
 - 인스턴스 = 설계도에 따라 준비한 메모리
 - 레퍼런스 = 변수가 있는 메모리 위치

*인스턴스와 레퍼런스



*레퍼런스 배열과 인스턴스
 - Member[] arr = new Member[3]
 - 인스턴스 3개를 만드는 게 아니라, 레퍼런스 3개를 만드는 것

 - 선언 후, 지정해야 함 Ex : arr[0] = new Member();
